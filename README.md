## 🚀 Como Executar o Projeto Localmente

1. **Clone o repositório:**
    ```bash
    git clone https://github.com/ShiroiCrypto/WebEmily.git
    ```
2. **Navegue até o diretório do projeto:**
    ```bash
    cd WebEmily
    ```
3. **Abra o arquivo `index.html` no seu navegador preferido:**
    - Para Windows:
        ```bash
        start index.html
        ```
    - Para macOS:
        ```bash
        open index.html
        ```
    - Para Linux:
        ```bash
        xdg-open index.html
        ```
